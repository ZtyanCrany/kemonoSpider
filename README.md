# kemonoSpider
获取kemono图片的C#爬虫脚本<br>
使用方法为输入当前条目下的链接，如"https://kemono.party/fanbox/user/{user_id}/post/{post_id}/" 即可原画质下载全部图片（支持gif，但不支持视频）。
